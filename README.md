# MRI-tumor-Detection-app

**Live demo:** https://mri-tumor-detection-app-530180677158.us-central1.run.app

A web app that takes an uploaded MRI image, runs it through your trained Keras model, and returns a tumor classification (glioma, meningioma, pituitary, or no tumor) with confidence scores — built as a FastAPI backend serving both the prediction API and a simple upload/results frontend, deployed as a single online app on Google Cloud Run.

> Educational project — not a diagnostic tool. Do not use for real medical decisions.
